### 💡 使用方法

### 📘 補足

* これらの `.ipynb` はそれぞれ対応する `.py` の中身をそのままセル化しています。
* `numpy` 以外の外部ライブラリは不要です。
* 教材目的でマルコフ連鎖の挙動・定常分布を観察することができます。
